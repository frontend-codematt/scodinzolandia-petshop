COME APRIRLO (IMPORTANTE)
Il sito carica i componenti e i dati via JavaScript, quindi NON funziona
facendo doppio clic sui file (il browser blocca i caricamenti su file://).
Va aperto tramite un piccolo server web locale. Due modi semplici:

1) Con Python (gia presente su Mac/Linux):
   - Apri il Terminale dentro questa cartella
   - Esegui:  python3 -m http.server 8000
   - Apri nel browser:  http://localhost:8000/Scodinzolandia%20Home.dc.html

2) Con Node.js:
   - npx serve .
   - Apri l'indirizzo mostrato a terminale

In alternativa, carica l'intera cartella su un qualsiasi hosting web
(Netlify, Vercel, il tuo spazio FTP) e il sito sara online.
